# Teditor

**Teditor** is a terminal-based text editor similar to *nano*, built with Python 🐍

### **NOTE**: This project is Work In Progress, expect bugs.

## 🗂️ Supported Languages

- C
- C++
- C#
- Python
- Brainfuck
- CSS
- HTML
- JavaScript
- Go
- Java
- JSON
- Kotlin
- Markdown *(written using teditor)*
- Rust
- Bash
- Vim
